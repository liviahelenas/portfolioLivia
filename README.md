# ✨ Portfólio — Lívia Helena
## 📌 Sobre o projeto

Este projeto é um portfólio pessoal desenvolvido para apresentar a trajetória, habilidades e projetos como designer e desenvolvedora front-end.

🔗 **[Acesse o projeto ao vivo](https://liviahelenas.github.io/portfolioLivia/main/index.html)**

O site foi pensado como uma experiência visual delicada e moderna, unindo:

* UX/UI Design
* Desenvolvimento Front-End
* Direção de arte digital
* Interatividade e identidade visual

A proposta do projeto é transmitir personalidade através de uma interface responsiva, estética e funcional, focada na experiência do usuário.
---
## 🎨 Conceito Visual

O design do portfólio segue uma identidade visual suave e criativa, utilizando:

* Gradientes suaves
* Glassmorphism
* Elementos autorais
* Hierarquia visual bem definida
* Microinterações e animações

Toda a estrutura foi construída pensando em clareza, navegação intuitiva e storytelling visual.
